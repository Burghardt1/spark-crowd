object Kajino

Provides functions for transforming an annotation dataset into a standard label dataset using the Kajino algorithm

This algorithm only works with com.enriquegrodrigo.spark.crowd.types.BinaryAnnotation datasets

Example: 

  1. result: KajinoModel  = Kajino(dataset, annotations)
Version

0.1

See also

Kajino, Hiroshi, Yuta Tsuboi, and Hisashi Kashima. "A Convex Formulation for Learning from Crowds." Transactions of the Japanese Society for Artificial Intelligence 27.3 (2012): 133-142.
